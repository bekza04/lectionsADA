git init - создает гит репозиторий из текущей папки (пишется 1 раз в самом начале)
git add . - (либо вместо точки люой файл из этой директории) - добавляет файлы и папки для 
отслеживания в индекс (эту команду пишем каждый раз перед коммандой commit)
git commit -m "сообщение" - она создает новую версию если были изменения в отслеживающих файлах.


аолный путь от создания локального репозитоия до отправки на удаленный 
1) создаем новый репозиторий на гитхаб 
2) git init - инициализируем локальный репозитоий после команды в директории должен 
появиться файл .git для его прсомотра можно выполнить ls -a
3) git add.  добавляет все файлы в индекс гита
4) git commit -m "comment"  - коммитим все изменения и сохранения
5) устанавлиеваем связь через ключ SSH
 1) генерирует ssh ключ через команду ssh-keygen при создании ключа, в сообщении будет указан путь котрому он сохранение
 2) звходим в папку  .ssh и копируем содержимое файла  id_rsa.pub (cat)
 3) заходим на гитхаб и переходим в настройки 
 4) заходим в пункт ssh and gpg keys и нажимаем на new ssh key 
 5) пишем назване для ключа(может быь любым) и ниже в полс=е для ключа вставляем ключ который скопирован из  id_rsa.pub
6) git remote add origin <SSH-ключ с GitHub/ либо с HTTPS ссылку устанавливаем соединение 
7) git push origin main/master - командая которая отправляет все изменения на удаленный репрозиторий на гитхаб
